# ⚙️ Rotas

> **GET**
>
> Retorna somente um NCM específico
>
> ```text
> .../api/ncm/<numero_do_ncm>
> ```


> **GET**
>
> Retorna todos os dados dos produtos registrados na TIPI
>
> ```text
> .../api/ncm/all
> ```

## 🌐Template

> **GET**
>
> Renderiza uma página HTML com todos os dados da tabela TIPI
>
> ```text
> .../
> ```
